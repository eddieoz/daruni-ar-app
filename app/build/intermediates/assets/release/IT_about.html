O aplicativo mostra alguns dos produtos da loja Terceiraidade.com em 3D, utilizando uma tecnologia chamada Realidade Aumentada para gerar um holograma com ultra definição.

Funcionalidades

  • Visualize os produtos como se estivessem ao alcance das mãos
  • Observe todos os ângulos, encaixes e material
  • No Menu, habilite a Descrição para mais informações

Baixe e imprima o objeto-alvo DH3D

Antes de iniciar, no browser do seu computador, baixe e imprima o arquivo PDF deste endereço: http://3ida.de/pedras3d (pode ser preto e branco)
Dica: Quanto maior, melhor!

Instruções

  • Clique em Iniciar
  • Aponte a Câmera para o Objeto-alvo DH3D
  • Toque na tela para melhorar o Foco
  • Dois toques no objeto para abrir a página do produto na loja Terceiraidade.com
  • Deslize a tela para a direita para abrir o menu de opções
