O aplicativo Daruni mostra alguns dos produtos da loja Terceiraidade.com em 3D, utilizando uma tecnologia chamada Realidade Aumentada.

Funcionalidades

  • Visualize os produtos como se estivessem ao alcance das mãos
  • Observe todos os ângulos, encaixes, material
  • No Menu, habilite a Descrição para mais informações

Baixe e imprima o objeto-alvo DH3D

Antes de iniciar, no seu computador imprima o arquivo PDF deste link: http://3ida.de/pedras3d (pode ser preto e branco)

Instruções

  • Clique em Iniciar
  • Aponte a Câmera para o Objeto-alvo DH3D
  • Toque na tela para melhorar o Foco
  • Dois toques na tela para abrir a página do produto
  • Deslize a tela para a direita para o menu de opções
